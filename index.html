How to use
  1. Enter ally code + event ID above, click Copy script
  2. Open swgoh.gg in your browser (any page, must be logged in)
  3. Open DevTools console (F12 → Console) and paste the script
  4. Click the orange START SCOUT FETCH button the script injects (a real click is required — swgoh.gg blocks script-only navigation)
  5. A JSON file downloads automatically — come back here and import it
  6. Click → Load into Planner to fill your Planner slots
